# features

Módulos organizados por domínio de negócio (não por tipo de arquivo). Cada subpasta agrupa os componentes, hooks e lógica específicos daquela feature: `home`, `about`, `blog`, `contact`, `solutions`, `projects`, `technologies`.

Código compartilhado entre features pertence a `components/`, `hooks/` ou `services/` na raiz de `src/`.
